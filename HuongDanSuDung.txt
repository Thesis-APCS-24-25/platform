## Build and run

Execute:
```bash
sh ./scripts/build.sh
cd ..
```

Accessing the URL http://host.docker.internal:8087 will lead you to the app in development mode.

Then go to http://localhost:8080, sign up a new user, and create a workspace.
Password strength is not enforced, email is not verified, and the user can log in with any password.

Note: login with email code is currently not supported, so use login with password.
